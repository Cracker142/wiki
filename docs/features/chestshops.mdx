# Честшопы

Для того, чтобы торговать в автономном режиме вам нужно будет использовать механику «честшопов». Иными словами, сундук, на который поставили табличку.

## **Создание честшопа**

### **Создание магазина на продажу**

1. Первую строку **пропускаете**. *(там будет ваш ник)*
2. Во второй пишите **[cs]** или **[ChestShop]**
3. В третьей **S x** *("s" = sell)*, где *"х"* - количество монет.
4. Четвертую строку **пропускаете**.

![](https://sun9-65.userapi.com/impf/3plBTbryX663nVJsn_Hnbh1hMJ4IzNqiAWqY4w/blRZi7p1QRU.jpg?size=420x220&quality=96&sign=3dbe506ffedb47dd14af5e75a079c392&type=album)

5. После создания честшопа нужно будет нажать **ПКМ** предметом, который вы продаете, по табличке. Далее откроется меню, где можно выбрать количество товара: 

![](https://sun9-55.userapi.com/impf/E__O-Fvh1tai2t2zG9pieGi-pgmJ2xLzIGDxjQ/_I4_96mUDiI.jpg?size=420x176&quality=96&sign=1740668bf26ecb1bef5905db8f92b27a&type=album)

### **Создание магазина на покупку**

1. Первую строку **пропускаете**. *(там будет ваш ник)*
2. Во второй пишите **[cs]** или **[ChestShop]**
3. В третьей **B x** *("b" = buy)*, где *"х"* - количество монет.
4. Четвертую строку **пропускаете**.

![](https://sun9-74.userapi.com/impf/2ZvQufdWpvurHmYoJQvZIQCOTjwe0co1TE4q3A/OaWSnmJeaVM.jpg?size=420x225&quality=96&sign=561d79515123dad5130441c9fd6aac50&type=album)

5. После создания честшопа нужно будет нажать **ЛКМ** предметом, который вы покупаете, по табличке. Далее откроется меню, где можно выбрать количество товара: 

![](https://sun9-79.userapi.com/impf/BSjr_1-IbuA5VqYGwMBM5qdSX_pmWsWThEK50Q/VV1_Z3m33Ts.jpg?size=420x170&quality=96&sign=90a3c5acc39730ddf8b069e2e242a5fc&type=album)

### **Создание магазина на обмен**

1. Первую строку **пропускаете**. *(там будет ваш ник)*
2. Во второй пишите **[cs]** или **[ChestShop]**
3. Третью строку **пропускаете**.
4. Четвертую строку **пропускаете**.

![](https://sun9-56.userapi.com/impf/WsYsv0S-GeQcp1Li-5Uk2xrV0yneW_oQFE4xtQ/UKHbJbpRGWk.jpg?size=420x220&quality=96&sign=bd90fcdf5dae85c2bd64014d571ca299&type=album)

5. После создания честшопа нужно будет нажать **ЛКМ** предметом, который вы продаете, а затем **ПКМ** предметом, который вы покупаете. 


### **Взаимодействие игроков с готовым честшопом**

* Для того, чтобы убедиться в правильности, вы можете нажать **ПКМ** по табличке с уже готовым честшопом:

![](https://sun9-21.userapi.com/impf/ZDiyeJCAex-o-t0C8ozD7UX8BL39mHdhzzxtnA/zIoq678mYDc.jpg?size=420x235&quality=96&sign=fdd30ea6c90f6c9feb7587ed19188055&type=album)

* Если продаваемый *(с точки зрения покупателя)* предмет не установлен, то идет продажа первого предмета, который продавец положил в сундук. 

> Таким образом можно без проблем продавать алкоголь, наркотики и другие предметы.
